# Pokédex

A modern, fast, and responsive Pokédex application built with vanilla JavaScript and powered by the PokéAPI. The project focuses on clean architecture, performance, and an intuitive user experience without relying on frameworks or build tools.

## Features

* Browse and search Pokémon
* Detailed Pokémon information
* Responsive design for desktop and mobile devices
* Dynamic data fetched from the PokéAPI
* Fast loading and optimized user experience
* Clean and maintainable code structure

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* PokéAPI

## Usage

* Search Pokémon by name
* Browse through the Pokémon collection
* View detailed information such as:

  * Type
  * Height
  * Weight
  * Stats
  * Abilities
