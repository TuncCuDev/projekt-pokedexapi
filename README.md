 Pokédex Web App

A modern, fast, and clean Pokédex built with vanilla JavaScript, powered by the PokéAPI. Focus on performance, UX, and maintainable architecture – no frameworks, no build tools.

 Features

    Search Pokémon by name (with minimum character threshold)

   Filter by: • Type • Generation • Favorites

   Sorting • By ID • Name A–Z / Z–A

   Favorites system • Mark Pokémon as favorites • Persisted via localStorage

   Detailed modal view • About • Base stats • Evolution chain • Shiny variants

   Evolution view • Complete evolution path • Current Pokémon highlighted • Click any evolution to navigate instantly

   Progressive loading • Paginated API loading • “Load more” behavior
   
   Custom loading overlay • Animated Pokéball loader

   Keyboard navigation • Arrow keys (prev / next) • Escape to close modal

   Responsive design • Optimized for desktop & mobile
